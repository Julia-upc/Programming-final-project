◆ processAssets()

None webscraping.processAssets ( webdriver.ChromeOptions  options)

It processes all the assets of interest, all of which have been detailed in the project statement.

The datasets are processed one at a time using the 'scrapDataFromUrlToDf' function. The DataFrames are then stored in CSV files.
